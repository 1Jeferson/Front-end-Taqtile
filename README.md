# Desafio Taqtile

## Project's Description

Front-end project developed through a selection process at Instaq.

This project is a web application designed to manage users, including functionalities such as user creation, listing, and detailed view.

#### User Management:

- User listing;
- Display of complete details for each user;
- Form for creating new users;
- Integration with Apollo Client to perform GraphQL queries and mutations.

## Steps to run and debug

- Clone the repository

```
git clone https://github.com/1Jeferson/Front-end-Taqtile.git
```

- Change directory

```
cd Front-end-Taqtile
```

- Install the dependencies

```
npm install
```

- Start the project

```
npm run dev
```
